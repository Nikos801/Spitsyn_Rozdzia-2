Informacje o JavaScript(Spitsyn Nikita 3-K)

1.Cztery sposoby wyprowadzania danych w JavaScript

1)console.log() – wyświetlanie danych w konsoli przeglądarki.

2)alert() – wyświetlanie komunikatu w oknie alertu.

3)document.write() – wypisywanie treści bezpośrednio w dokumencie HTML.

Czym jest funkcja w JavaScript? (omówienie w 4 punktach)

1)Definicja – Funkcja to blok kodu, który można wywoływać wielokrotnie.

2)Deklaracja – Można ją zadeklarować przy użyciu słowa kluczowego function lub jako funkcję strzałkową (=>).

3)Parametry i zwracanie wartości – Funkcja może przyjmować argumenty i zwracać wynik za pomocą return.

4)Przykład użycia:function dodaj(a, b) { return a + b; } console.log(dodaj(3, 5)); // 8

3. Metody pobierania elementów w DOM (Tabela)

Metoda Opis Przykład
getElementsByName Pobiera wszystkie elementy o danej nazwie (atrybut name).. document.getElementsByName("username")
getElementById Pobiera jeden element o określonym identyfikatorze (id). document.getElementById("header")
getElementsByTagName Pobiera wszystkie elementy danego typu (tag name).). document.getElementsByTagName("p")

4. Omówienie właściwości: innerHTML innerText className.

innerHTML

Umożliwia odczyt i modyfikację kodu HTML wewnątrz elementu. Przykład:document.getElementById("content").innerHTML = "Nowa zawartość";

innerText

Umożliwia odczyt i modyfikację tylko widocznego tekstu wewnątrz elementu (ignoruje znaczniki HTML). Przykład:document.getElementById("content").innerText = "Zwykły tekst";

className

Służy do odczytu i zmiany klasy CSS elementu. Przykład:document.getElementById("content").className = "nowa-klasa";
