Informacje o JavaScript(Spitsyn Nikita 3-K)

1. Kiedy powstał JavaScript?

JavaScript powstał w grudniu 1995 roku.

2. Dlaczego jest to język skryptowy?

JavaScript jest językiem skryptowym, ponieważ jest interpretowany i wykonywany bezpośrednio w przeglądarce, bez potrzeby kompilacji.

3. Gdzie jest wykonywany JavaScript (klient lub serwer)?

JavaScript może być wykonywany zarówno po stronie klienta (w przeglądarce), jak i po stronie serwera (np. z użyciem Node.js).

4. Dlaczego JavaScript to bezpieczne narzędzie?

JavaScript jest stosunkowo bezpieczny dzięki izolacji w tzw. "sandboxie" przeglądarki, ograniczając dostęp do zasobów lokalnych użytkownika.

5. Czy jest to język obiektowy, jeśli tak, to dlaczego?

Tak, JavaScript jest językiem obiektowym, ponieważ umożliwia tworzenie obiektów z właściwościami i metodami oraz wspiera programowanie prototypowe.

6. Zalety JavaScript

  • Wykonuje się bezpośrednio w przeglądarce, co umożliwia szybkie interakcje.
  • Jest wszechstronny i może działać po stronie klienta oraz serwera.
  • Ma szerokie wsparcie społeczności, co ułatwia rozwój aplikacji.

7. Framework to:

Framework to zestaw narzędzi i bibliotek ułatwiający tworzenie aplikacji poprzez strukturyzację kodu i uproszczenie pracy programisty.

8. Do aplikacji mobilnych można użyć Framework

Tak, można użyć frameworków takich jak React Native, Ionic czy NativeScript.

9. Aplikacje responsywne to:

Aplikacje, które dostosowują się do różnych rozmiarów ekranów, zapewniając wygodne korzystanie na różnych urządzeniach.

10. Przykład kodu z przyciskiem otwierającym nowe okno:

Ten przycisk otwiera nowe okno z wybranym adresem (w tym przypadku obraz.html), pod warunkiem, że funkcja WinOpen() jest wcześniej zdefiniowana.

11. Przykład kodu otwierającego okno za pomocą window.open:

        window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

Ten kod otwiera nowe okno o nazwie okienko z wyłączonym paskiem narzędzi i menu, w rozmiarze 280x160 pikseli.

12. Instrukcja window.close():

Instrukcja window.close() zamyka aktualne okno lub zakładkę przeglądarki, jeśli zostało otwarte przez JavaScript.
