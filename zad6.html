Informacje o JavaScript(Spitsyn Nikita 3-K)

1) Rodzaje okienek

alert: Wyświetla komunikat informacyjny. Blokuje interakcję do czasu zamknięcia. javascript alert("To jest alert!"); confirm: Wyświetla komunikat z opcjami „OK” i „Anuluj”. Zwraca true lub false. javascript let decyzja = confirm("Kontynuować?"); prompt: Pobiera dane tekstowe od użytkownika. javascript let imie = prompt("Jak masz na imię?");

Konwersji liczb

JJawna konwersja: Number(): Przekształca tekst na liczbę. javascript Number("123"); // 123 parseInt(), parseFloat(): Zamienia tekst na liczbę całkowitą lub zmiennoprzecinkową. javascript parseFloat("123.45"); // 123.45 Niejawna konwersja: Dokonywana automatycznie, np.: javascript "123" * 2; // 246

3)Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad:

Nazwa musi zaczynać się od litery, _ lub $. Nie może zawierać spacji ani znaków specjalnych. Nie może być słowem zarezerwowanym, np. let, if.

4) Typy zmiennych w JavaScript

let: Zmienna o zasięgu blokowym, można ją zmieniać. const: Stała, nie można zmienić jej wartości. var: Starszy sposób deklarowania, ma zasięg funkcyjny.
